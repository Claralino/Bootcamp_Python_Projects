
Só pode ocorrer operações com dados do mesmo tipo 

ADIÇÃO
    +

SUBTRAÇÃO
    -

MULTIPLICAÇÃO
    *

DIVISÃO
    /
    Sempre resulta em um float

POTENCIA
    **

Ordem que acontece a conta
    1. ()
    2. **
    3. *  /  -> o mais pra esquerda priorizado
    4. +  -  -> o mais pra esquerda priorizado

ARREDONDAMENTO
    usar a função round():
        round(8 / 3)
    pode mostrar quantas casas decimais voce quer:
        round(8 / 3, 2) -> arredonda pra 2 casas decimais 
    ou usar //:
        8 // 3 -> recebe um int direto

MACETES
    para abreviar as operações:
        result = 4 / 2
        result /= 2 == result = result / 2

        result = 4 + 2
        result += 2 == result = result + 2
