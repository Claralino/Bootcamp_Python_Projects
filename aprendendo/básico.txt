DATA TYPES 
    String  
        sequência de caracteres com "" 
        é possível pegar um caractere dessa forma:
            "hello"[0] -> vai pegar o caractere h
    
    Integer
        numeros inteiros
    
    Float
        numeros com casas decimais
        sempre usa . 
    
    Boolean
        só tem 2 valores, True ou False
        
    Transformar os tipos de dados
        para transformar um tipo de dado em outro:
        str(dado) -> virou string
        float(dado) -> virou Float
        int(dado) -> virou int

F STRING 
    serve para misturar diferentes tipos de datas
    nao precisa converter nenhuma
    botar f em frente à string, e botar o diferente tipo de data dentro de chaves {}:
        f"sua nota é: {nota}" -> nota é int, mas aparece mesmo assim
