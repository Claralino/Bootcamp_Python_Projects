LISTS

    serve para armazenar mais de um dados
    podem armazenar diferentes tipos de dados juntos:
        lista = [item1, item2, item3, item4, item5, item6]
    para achar um item na lista apenas botar seu índice do lado:
        print(lista[0]) -> item1
        print(lista[-1]) -> item6
    para add um item ao final da lista, usar a função append (existem varias funções pela internet):
       lista.append(item7) -> é add no final da lista

NESTED LISTS

    listas dentro de listas:
        fruits = [item1, item2, item3, item4,]
        veggies = [item5, item6, item7, item8]
        nested = [fruits, veggies] -> é uma lista com as duas outras listas

é possivel checar e ver se um elemento esta dentro da lista, tipo: if "apple" not in lista:
